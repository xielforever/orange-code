# Orange Code v1.0.0 发布说明

## 概述

Orange Code `v1.0.0` 标志着本项目作为独立个人二次开发工具的正式启动。本项目基于开源项目 [Orange Code](https://github.com/instructkr/orange-code) 的优秀架构（受 Claude Code 启发）进行重构与演进。本次发布的核心在于提供一个可用的本地 CLI 体验：包括交互式会话、非交互式提示（prompts）、工作区工具、配置加载、会话管理、插件机制以及本地代理/技能发现。

## 亮点

- 以安全的 Rust 实现作为主要的产品形态
- `orange` CLI，用于交互式和单次编码代理工作流
- 内置感知工作区的工具，支持：终端 shell、文件操作、搜索、网页抓取/搜索、待办事项（todo）跟踪以及笔记本更新
- 提供用于状态查看、上下文压缩、配置检查、会话管理、差异（diff）/导出和版本信息的斜杠命令（Slash-command）
- 本地插件、代理（agent）和技能的发现/管理界面
- OAuth 登录/注销机制以及模型/提供商选择功能

## 安装与运行

此版本目前主要面向从源码构建：

```bash
cargo install --path crates/orange-cli --locked
# 或者
cargo build --release -p orange-cli
```

运行:

```bash
orange
orange prompt "总结这个存储库"
```
